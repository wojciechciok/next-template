- Tailwind CSS IntelliSense

## Formatting on save

1. `Ctrl + Shift + P`
2. Choose `Preferences: Open Settings (JSON)`
3. Paste the following to the object:

`"editor.defaultFormatter": "esbenp.prettier-vscode", "editor.formatOnSave": true`
